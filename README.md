#  Projeto Pedido do Cliente -  ☕︎

Projeto em que um cliente informa seus dados(nome, e-mail, data de nascimento), o Status de seus pedidos e seus pedidos.   
O programa verifica se a Data de Nascimento, E-mail e dados do Pedido informado são válidos

&nbsp; •  &nbsp; Composição de Obejetos  
&nbsp; •  &nbsp; Encapsulamento  
&nbsp; •  &nbsp; Data / Hora   
&nbsp; •  &nbsp; Métodos   
&nbsp; •  &nbsp; Relacionamento entre Classes   
&nbsp; •  &nbsp; Relacionamentos de Agregação
