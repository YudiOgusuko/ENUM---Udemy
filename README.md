#  Projeto Pedido do Cliente -  ☕︎

Projeto em que um cliente informa seus dados pessoais(nome, e-mail, data de nascimento) e os Dados e Status de seus pedidos.   
O programa verifica se os dados informado são válidos, após isso é exibido todas as informações cadastradas.

## **Conteúdo**

&nbsp; •  &nbsp; Enumeração   
&nbsp; •  &nbsp; Composição de Obejetos  
&nbsp; •  &nbsp; Encapsulamento  
&nbsp; •  &nbsp; Data / Hora   
&nbsp; •  &nbsp; Métodos   
&nbsp; •  &nbsp; Relacionamento entre Classes   
&nbsp; •  &nbsp; Relacionamentos de Agregação    

## **Tecnologias**
